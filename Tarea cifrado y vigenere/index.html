Práctica Descripción
Cesar Este JavaScript implementa un cifrado Vigenère con la letra ñ. Un módulo define métodos de cifrado y descifrado, utilizando un alfabeto de 27 caracteres. Una función determina el índice de cada letra. Las funciones codificar y decodificar validan campos, aplican desplazamientos cíclicos, y devuelven resultados. Por último, reiniciar limpia campos.
vigenere Este código implementa un sistema de cifrado y descifrado mediante el cifrado de Vigenere, utilizando un alfabeto extendido que incluye la letra 'ñ'. Además, integra funciones para manejar entradas de texto y clave desde elementos HTML (campos de formulario o similares)
